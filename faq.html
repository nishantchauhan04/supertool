Frequently Asked Questions

Find answers to common questions about SuperTool

General Questions

What is SuperTool?

SuperTool is a free, all-in-one online toolkit that provides 13+ powerful tools for everyday tasks. From image conversion to financial calculators, all tools run directly in your browser without requiring any software installation.

Is SuperTool really free?

Yes! SuperTool is 100% free with no hidden costs, subscriptions, or premium tiers. All features are available to everyone at no charge. We support the service through non-intrusive advertising.

Do I need to create an account?

No account required! All tools are instantly accessible without any registration or login. Simply visit the website and start using any tool immediately.

What browsers are supported?

SuperTool works on all modern browsers including Chrome, Firefox, Safari, Edge, and Opera. We recommend using the latest version of your browser for the best experience.

Privacy & Security

Is my data safe?

Absolutely! All tools process data entirely in your browser. Your files and information never leave your device. We don't upload, store, or have access to any of your data.

Do you store my files?

No, we never store your files. All processing happens locally on your device. Once you close the browser tab, all data is automatically cleared from memory.

Can I use SuperTool offline?

Once the page is loaded, most tools will work offline since they run in your browser. However, you'll need an internet connection to initially load the website.

Do you track my usage?

We use basic analytics to understand how visitors use our site (like page views and popular tools), but we never track or store the actual content you process. See our Privacy Policy for details.

Tool-Specific Questions

What file formats are supported?

Different tools support different formats:

  • Image Tools: JPG, PNG, WebP, GIF
  • PDF Tools: PDF files
  • Text Tools: Plain text, JSON, Base64

Is there a file size limit?

File size limits depend on your device's memory and browser capabilities. Generally, files up to 50-100MB work well. Very large files may take longer to process or may not work on devices with limited memory.

Can I process multiple files at once?

Yes! Tools like Image to PDF Converter and Image Compressor support batch processing. You can upload and process multiple files simultaneously.

Why is my PDF/image quality reduced after compression?

Compression reduces file size by removing some data. Our tools are optimized to maintain good quality while reducing size. You can adjust compression settings to balance between file size and quality.

How accurate are the calculators?

Our EMI Calculator and Rent vs Buy Calculator use standard financial formulas and provide accurate estimates. However, actual results may vary based on specific terms, fees, and conditions from lenders or landlords.

Technical Questions

Why is a tool not working?

If a tool isn't working, try these steps:

  • Refresh the page and try again
  • Clear your browser cache
  • Try a different browser
  • Check if your file format is supported
  • Ensure your file isn't corrupted

Can I use SuperTool on mobile devices?

Yes! SuperTool is fully responsive and works on smartphones and tablets. Some tools may have limited functionality on older mobile devices due to processing power constraints.

Does SuperTool work on slow internet connections?

Yes! Once the page loads, all tools work locally on your device without requiring internet. Only the initial page load requires an internet connection.

Is there an API available?

Currently, we don't offer an API. All tools are designed for direct browser use. However, our code is open source on GitHub if you'd like to integrate similar functionality into your own projects.

Support & Feedback

How can I report a bug?

If you encounter a bug, please visit our GitHub repository and create an issue with details about the problem, including your browser version and steps to reproduce the issue.

Can I request a new tool?

Absolutely! We're always looking to add useful tools. Share your suggestions on our GitHub repository or through our contact channels.

Can I contribute to SuperTool?

Yes! SuperTool is open source. You can contribute code, report bugs, suggest features, or help improve documentation on our GitHub repository.

How can I support SuperTool?

You can support us by:

  • Sharing SuperTool with friends and colleagues
  • Leaving feedback and suggestions
  • Contributing to the open source project
  • Allowing non-intrusive ads (they help keep the service free)
